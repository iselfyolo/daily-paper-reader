## 摘要
模型计数（#SAT）是概率推理的核心，是一个基础且属于 #P-完全（#P-complete）的问题。在这项工作中，我们研究了增量式模型计数，即对一系列结构相似的公式进行计数。我们提出了一种通过持久化缓存机制分摊计算的方法，在多次求解器调用之间保留组件数据，以避免冗余搜索。此外，我们研究了适用于该场景的分支启发式算法。我们专注于论证（argumentation）和软核（soft core）问题，在这些问题中，增量式模型计数是自然而然的需求。实验表明，与现有的模型计数器相比，我们的方法提高了性能，突显了在动态环境中结构感知重用的能力。

## Abstract
Model counting ($\#\text{SAT}$) is a fundamental yet $\#\text{P}$-complete problem central to probabilistic reasoning. In this work, we address \textit{incremental model counting}, where sequences of structurally similar formulas must be counted. We propose an approach that amortizes computation via a persistent caching mechanism, retaining component data across solver calls to avoid redundant search. Additionally, we investigate branching heuristics adapted for this setting. We focus on the problems of argumentation and soft core, for which incremental model counting is natural. Experiments demonstrate that our method improves performance compared to current model counters, highlighting the capability of structure-aware reuse in dynamic environments.

这篇论文《Efficient Incremental #SAT via Cross-Instance Knowledge Reuse》由来自以色列公开大学和法国 CRIL 实验室的研究者共同完成。以下是对该论文的结构化总结：

### 1. 核心问题与整体含义
*   **研究动机**：模型计数（#SAT）是计算满足布尔公式的解的数量，属于 #P-完全问题。传统的 #SAT 求解器通常将每个实例视为孤立的，即使在一系列公式之间只有微小的改动（如在动态论证或软核计算中），求解器也会从头开始计算。
*   **核心问题**：如何在处理一系列结构相似的公式序列时，通过重用之前的计算知识（知识重用）来减少冗余搜索，从而提高增量式模型计数的效率。

### 2. 方法论
论文提出了一个名为 **d4-dyn** 的增量框架，核心思想包括：
*   **持久化缓存（Persistent Caching）**：
    *   **显式表示**：为了确保在公式变化时的正确性，放弃了传统求解器（如 d4, SharpSAT）中依赖静态公式结构的压缩缓存方案，转而采用完全显式的子公式表示。
    *   **跨实例重用**：在不同的求解调用之间保留缓存，如果新实例中出现了之前解决过的子问题，直接返回结果。
*   **延迟对称缓存（Lazy Symmetry Caching）**：
    *   通过计算文字出现的频率来生成规范序（Canonical Form），并利用置换（Permutation）检测变量重命名后的等价子问题，从而提高缓存命中率。
*   **启发式共享（Heuristic Sharing）**：
    *   **树分解（TD）重用**：计算初始公式的树分解，并在后续仅涉及子句删除的实例中重用该结构，以指导变量分支顺序。
    *   **确定性启发式**：倾向于使用 DLCS 而非 VSADS，因为确定性的分支策略能产生更稳定的搜索模式，更有利于缓存命中。

### 3. 实验设计
*   **应用场景**：
    1.  **动态论证（Dynamic Argumentation）**：计算论证框架演化过程中的完整扩展数量（1000 个增量步骤）。
    2.  **软核计算（Soft Core Computation）**：通过迭代删除子句来寻找满足模型计数阈值的最小子句集。
*   **数据集**：
    *   2023 年国际论证竞赛（ICCMA）的 264 个 AF 实例。
    *   包含电路、规划、随机 3-SAT 等 3040 个软核基准测试公式。
*   **对比方法**：
    *   **ganak**：2025 年模型计数竞赛冠军。
    *   **PBCount2**：现有的增量式伪布尔模型计数器。
    *   **d4**：原始的静态模型计数器。
    *   **消融实验**：对比了 NoShared（无共享）、Shared（共享缓存）、Shared+Sym（共享+对称检测）以及是否开启 TD 共享。

*   **实验规模**：论文涵盖了数百个论证框架序列和数千个软核计算实例，实验规模较大。

### 6. 主要结论与发现
*   **缓存重用的威力**：在动态论证任务中，Shared+Sym 模式相比 NoShared 模式实现了平均 **5000%** 的性能提升。
*   **启发式影响**：确定性的 **DLCS** 启发式在增量设置下优于 VSADS，因为它能产生更一致的搜索路径，从而提高缓存利用率。
*   **结构重用**：对于困难实例，重用**树分解（TD）**信息能显著减少搜索空间，尽管在简单实例上 TD 的初始计算开销可能不划算。
*   **对比结果**：d4-dyn 在大多数增量基准测试中优于 ganak 和 PBCount2，尤其是在处理长序列更新时。

### 7. 优点
*   **首创性**：这是首个在基于 DPLL 的模型计数中实现跨实例缓存和启发式重用的工作。
*   **结构感知**：通过对称检测和树分解重用，深入挖掘了公式间的结构相似性。
*   **实用性强**：针对论证和软核等实际增量问题提供了高效的解决方案。

### 8. 不足与局限
*   **内存压力**：为了保证正确性而采用的“显式缓存表示”比静态求解器的压缩表示更占内存，在处理极大规模公式时可能面临内存溢出风险。
*   **困难实例的边际效应**：在极难的实例上，知识重用的收益会递减，此时求解器的核心搜索能力（如 ganak 的预处理）可能比重用更重要。
*   **应用限制**：目前主要针对 CNF 公式，对于加权模型计数（WMC）等扩展问题的适用性仍需进一步研究（尤其是对称性处理部分）。
