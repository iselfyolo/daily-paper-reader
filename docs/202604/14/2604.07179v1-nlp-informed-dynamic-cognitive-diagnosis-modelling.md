## 摘要
数字学习平台在支持阅读能力发展的同时，也产生了丰富的日志文件和题目级文本内容。利用这些数据，本研究提出了一种动态认知诊断模型（CDM）框架，该框架结合了源自文本的语义信息，以辅助 Q 矩阵的估计。我们构建了题目文本和选项的题目级语义表示，并利用这些表示为 Q 矩阵定义了一个信息先验。该方法将源自文本的信号视为题目复杂度和认知需求的代理指标，以数据驱动的方式引导题目与技能的映射。所提出的框架在贝叶斯框架下联合估计潜在技能掌握情况、题目参数以及随时间变化的迁移动态。我们将该模型应用于数字阅读补充工具 Boost Reading 的数据，重点关注学生词汇和理解能力的发展。通过与不含文本信息的基准模型进行对比，结果表明，源自文本的先验能够提高 Q 矩阵的恢复效果（特别是在仅靠作答数据难以识别的情况下），并能改善不同场景下的其他模型参数。本研究实现了自然语言处理与动态 CDM 的创新整合，为数字学习环境下的技能习得和题目-技能关系建模提供了一种数据驱动的方法。

## Abstract
Digital learning platforms are increasingly used to support reading development while generating rich log files and item-level textual content. Using these data, this study proposes a dynamic cognitive diagnostic modelling (CDM) framework that incorporates text-derived semantic information to inform the estimation of the Q-matrix. We construct item-level semantic representations of question text and response options, and use these representations to define an informative prior on the Q-matrix. This approach treats text-derived signals as proxies for item complexity and cognitive demands, guiding the item-skill mapping in a data-driven manner. The proposed framework jointly estimates latent skill mastery profiles, item parameters, and transition dynamics over time within a Bayesian framework. We apply the model to data from Boost Reading, a digital reading supplement, focusing on students' vocabulary and comprehension skill development. We compare the proposed framework with a baseline model without any text information and show that the text-derived prior can improve Q-matrix recovery, particularly in settings where response data alone provide limited identification, as well as other model parameters for varying scenarios. This study provides a novel integration of natural language processing and dynamic CDMs, offering a data-driven approach to modelling skill acquisition and item-skill relationships in digital learning environments.

#### 1. 核心问题与整体含义（研究动机和背景）
认知诊断模型（CDM）在教育测量中用于评估学生对特定技能（属性）的掌握情况。其核心挑战在于 **Q 矩阵**（题目与技能的映射关系）的设定。传统上 Q 矩阵由专家人工标注，具有主观性且难以应对动态变化的数字学习环境。
本文的研究动机在于：在数字学习平台（如阅读训练系统）中，题目本身包含丰富的文本信息。作者提出利用 **自然语言处理（NLP）** 技术提取题目的语义特征，将其作为 Q 矩阵估计的**信息先验**，从而在缺乏专家标注或作答数据信息量不足的情况下，更准确地实现题目-技能的自动映射，并追踪学生随时间推移的技能演变。

#### 2. 方法论：核心思想与关键技术
该研究提出了一种 **NLP 告知的动态认知诊断框架**，主要包含以下三个部分：
*   **测量模型（DINA 模型）：** 建立学生潜在技能状态与观察到的作答准确率之间的联系，包含失误参数（Slipping）和猜测参数（Guessing）。
*   **结构模型（动态迁移）：** 采用一阶马尔可夫过程建模学生技能随时间的变化，并通过逻辑回归引入协变量（如初始能力、练习次数、反应时间等）来预测技能的获得或丧失。
*   **NLP 告知的 Q 矩阵先验：**
    *   **语义表示：** 使用 **SBERT**（Sentence-BERT）将题目题干、正确选项和干扰项转化为高维向量。
    *   **语义辨别力评分 ($\tau_j$)：** 计算题干与正确选项的相似度 ($S^+$) 与题干与干扰项平均相似度 ($S^-$) 之差。
    *   **信息先验：** 将 $\tau_j$ 引入 Q 矩阵条目的伯努利分布先验中。核心假设是：**语义辨别力越高（正确项与干扰项区分明显）的题目，其认知复杂度通常越低，对应的 Q 矩阵行越稀疏（涉及技能越少）。**
*   **参数估计：** 在贝叶斯框架下，使用 MCMC（吉布斯采样）在 `nimble` 环境中联合估计所有参数。

#### 3. 实验设计
*   **数据集：**
    *   **真实数据：** 来自美国广泛使用的数字阅读补充工具 **Boost Reading**。选取了两个游戏：*Idiomatica*（考察词汇技能）和 *Debate-a-ball*（考察理解技能）。样本包含 1,616 名学生在二年级和三年级两个时间点的数据。
    *   **模拟实验：** 设计了不同样本量 ($N=800, 1600, 2400$) 和题目数量 ($J=10, 20, 30$) 的多种组合场景。
*   **Benchmark（基准模型）：** 不包含文本信息的基准动态 CDM 模型（即 $\lambda=0$ 的模型）。
*   **对比指标：** Q 矩阵恢复准确率（ACC）、假阳性/假阴性率、后验包含概率（PIP）、属性掌握分类准确率（PAR/AAR）以及参数估计的均方根误差（RMSE）。

*   **软件工具：** 使用 R 语言的 `nimble` 包进行贝叶斯建模，使用 Python 的 `sentence-transformers` 库生成文本嵌入。

*   **实验规模：** 模拟研究进行了 **100 次独立重复实验**，并使用非参数 Bootstrap 方法（1,000 次重采样）计算标准误，以确保结果的稳定性。

#### 6. 主要结论与发现
*   **Q 矩阵恢复提升：** 引入文本先验显著提高了 Q 矩阵的恢复质量，尤其是在作答数据较少、模型识别性较弱的挑战性场景下。
*   **参数估计更稳定：** 相比基准模型，NLP 告知的模型在估计猜测、失误参数以及技能迁移参数时具有更低的误差（RMSE 和 MAE）。
*   **实证有效性：** 在 Boost Reading 数据中，模型成功识别出词汇和理解技能的增长趋势，并发现初始识字能力和练习行为（如重做次数）对技能掌握有显著预测作用。
*   **数据驱动的优势：** 证明了文本信号可以作为题目复杂度的有效代理，减少了对专家标注的依赖。

#### 7. 优点
*   **可解释性强：** 与纯黑盒的深度学习模型不同，该方法将 NLP 嵌入到经典的统计认知诊断框架中，保留了教育测量所需的参数可解释性。
*   **鲁棒性：** 采用贝叶斯框架，当文本信号与作答数据冲突时，模型能够以作答数据为主，避免了错误文本信息对结果的误导。
*   **动态追踪：** 能够同时处理 Q 矩阵估计和纵向技能追踪，适应现代自适应学习系统的需求。

#### 8. 不足与局限
*   **先验强度调节：** 控制文本信息影响强度的参数 $\lambda$ 的选择仍具有一定启发性，缺乏完全自动化的最优确定方法。
*   **语义假设的局限：** 模型假设“语义辨别力”与“认知复杂度”成反比，这在某些特定学科（如数学或逻辑推理）中可能并不成立。
*   **技能维度限制：** 实证研究仅关注了两个技能维度，对于具有数十个微细粒度技能的大规模 Q 矩阵，该方法的扩展性和计算压力尚待验证。
*   **文本依赖性：** 模型效果高度依赖于预训练语言模型（SBERT）的质量以及题目文本描述的详尽程度。
