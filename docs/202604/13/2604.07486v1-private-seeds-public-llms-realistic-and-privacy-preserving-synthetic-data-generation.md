## 摘要
大语言模型 (LLMs) 已成为合成数据生成的强大工具。一个特别重要的应用场景是生成私有文本的合成副本，这需要仔细平衡隐私与效用。我们提出了“逼真且保护隐私的合成数据生成”(RPSG)，该方法利用包括形式化差分隐私 (DP) 在内的隐私保护机制，以及包含个人信息的私有种子文本，来生成逼真的合成数据。与最先进的私有合成数据生成方法进行的全面对比实验表明，RPSG 在提供强有力隐私保护的同时，实现了对私有数据的高保真度。

## Abstract
Large language models (LLMs) have emerged as a powerful tool for synthetic data generation. A particularly important use case is producing synthetic replicas of private text, which requires carefully balancing privacy and utility. We propose Realistic and Privacy-Preserving Synthetic Data Generation (RPSG), which leverages privacy-preserving mechanisms, including formal differential privacy (DP); and private seeds, in particular text containing personal information, to generate realistic synthetic data. Comprehensive experiments against state-of-the-art private synthetic data generation methods demonstrate that RPSG achieves high fidelity to private data while providing strong privacy protection.

论文探讨了如何利用公共大语言模型（LLMs）生成既能保持原始数据分布特征（高保真度），又能严格保护隐私的合成文本数据。

*   **现有挑战**：
    *   **DP-SGD（差分隐私微调）**：计算成本高，且现代最强模型（如 GPT-4）多为 API 形式，无法直接应用梯度修剪。
    *   **提示词工程（Prompt-based）**：生成的合成数据往往缺乏原始数据的丰富性和实用性。
    *   **隐私风险**：模型容易记忆训练数据，导致成员推理攻击（MIA）和个人身份信息（PII）泄露。

#### 2. 方法论：RPSG 框架
论文提出了 **RPSG (Realistic and Privacy-Preserving Synthetic Data Generation)** 框架，核心思想是利用“私有种子”引导生成，并通过多阶段过滤确保隐私。
*   **阶段 1：抽象与情感对齐**：对私有种子进行 PII 过滤，使用抽象模型（如 BART）生成多个候选文本，并确保其情感极性与原文本一致。
*   **阶段 2：形式化差分隐私（DP）选择**：应用**高斯机制（Gaussian Mechanism）**，根据语义相似度（效用函数）从候选文本中选择一个满足 $( \epsilon, \delta )$-DP 的候选者。
*   **阶段 3：合成变体生成**：利用公共 LLM 对 DP 保护后的候选文本进行重写和变体生成。
*   **阶段 4：数据精炼（核心防御）**：
    *   训练一个**代理模型（Surrogate Model）**来捕捉生成数据的分布。
    *   利用**余弦相似度**过滤掉与私有种子过于接近的样本。
    *   利用**负对数似然（NLL）得分**过滤掉模型表现出高度自信（即可能被记忆）的样本，从而抵御成员推理攻击（MIA）。
