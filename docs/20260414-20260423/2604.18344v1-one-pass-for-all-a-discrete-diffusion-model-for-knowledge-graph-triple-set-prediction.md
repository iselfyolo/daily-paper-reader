## 摘要
知识图谱（KGs）由三元组组成，知识图谱补全（KGC）的目标是推断缺失的事实三元组。传统的 KGC 任务在给定三元组中一个或两个元素的情况下预测缺失元素。作为一个更符合实际的任务，三元组集合预测（TSP）任务旨在仅以观测到的知识图谱为条件，推断缺失三元组的集合，而不假设关于缺失三元组的任何部分信息。现有的 TSP 方法以逐个三元组的方式预测缺失三元组集合，难以捕捉预测三元组之间的依赖关系以确保一致性。为解决这一问题，我们提出了一种名为 DiffTSP 的新型离散扩散模型，将 TSP 视为一个生成任务。DiffTSP 通过离散扩散过程逐步向 KG 添加噪声，这是通过掩码关系边来实现的。随后，逆向过程在以不完整图为条件的背景下，逐渐恢复完整的 KG。为此，我们设计了一个结构感知去噪网络，该网络集成了关系上下文编码器和关系图扩散 Transformer，用于知识图谱生成。DiffTSP 能够以一次通过的方式生成完整的三元组集合，同时确保预测三元组之间的依赖关系。我们的方法在三个公开数据集上达到了最先进的性能。

## Abstract
Knowledge Graphs (KGs) are composed of triples, and the goal of Knowledge Graph Completion (KGC) is to infer the missing factual triples. Traditional KGC tasks predict missing elements in a triple given one or two of its elements. As a more realistic task, the Triple Set Prediction (TSP) task aims to infer the set of missing triples conditioned only on the observed knowledge graph, without assuming any partial information about the missing triples. Existing TSP methods predict the set of missing triples in a triple-by-triple manner, falling short in capturing the dependencies among the predicted triples to ensure consistency. To address this issue, we propose a novel discrete diffusion model termed DiffTSP that treats TSP as a generative task. DiffTSP progressively adds noise to the KG through a discrete diffusion process, achieved by masking relational edges. The reverse process then gradually recovers the complete KG conditioned on the incomplete graph. To this end, we design a structure-aware denoising network that integrates a relational context encoder with a relational graph diffusion transformer for knowledge graph generation. DiffTSP can generate the complete set of triples in a one-pass manner while ensuring the dependencies among the predicted triples. Our approach achieves state-of-the-art performance on three public datasets. Code: https://github.com/ADMIS-TONGJI/DiffTSP.

*   **核心问题**：传统的知识图谱补全（KGC）任务（如链接预测）通常假设已知三元组的部分信息（如给定头实体和关系预测尾实体），这在现实场景中往往不成立。更具挑战性且实用的任务是**三元组集合预测（TSP）**，即仅根据现有的不完整知识图谱，直接预测所有缺失的三元组集合。
*   **研究动机**：现有的 TSP 方法大多采用“逐个三元组预测”的模式，将每个三元组视为独立个体进行评分。这种方式忽略了三元组之间的**内在依赖关系和逻辑一致性**（例如：若存在“父亲”关系，则必然存在对称的“儿子/女儿”关系）。

### 2. 方法论：DiffTSP 模型
*   **核心思想**：利用**离散扩散模型（Discrete Diffusion Model）**，通过前向过程逐步掩码（Mask）关系边，再通过逆向过程从噪声中恢复出完整的三元组结构。
*   **关键技术细节**：
    *   **关系平衡任务生成**：采用元学习框架，将图划分为支持图（Support Graph）和查询图（Query Graph），并确保两者的关系分布一致，以增强模型的泛化能力。
    *   **前向扩散过程**：定义在离散状态空间上的马尔可夫链，通过转移矩阵 $Q_t$ 逐步将现有的关系边转变为“缺失（Mask）”状态。
    *   **结构感知去噪网络**：
        *   **关系上下文编码器（RCE）**：利用关系引导的消息传递机制捕捉局部结构特征。
        *   **关系图扩散 Transformer（RelDiT）**：引入关系感知注意力偏置（Relation-aware Attention Bias），通过 Transformer 架构捕捉长程的全局结构依赖。
    *   **训练目标**：优化加权二元交叉熵损失（$BCE_w$），通过逆频率加权解决知识图谱中极度稀疏和类别不平衡的问题。
    *   **采样流程**：从无边的初始状态开始，经过 $T$ 步迭代去噪，每一步根据去噪网络的预测概率进行离散状态采样。

### 6. 主要结论与发现
*   **性能领先**：DiffTSP 在所有数据集的 $F_{TSP}$ 指标上均达到了 SOTA（最先进）水平，尤其在处理具有复杂逻辑依赖的 CFamily 数据集时表现突出。
*   **依赖捕捉**：通过案例分析证明，DiffTSP 能够有效避免传统方法中常见的逻辑冲突（如同一实体既是“兄弟”又是“姐妹”），生成的集合内部一致性更高。
*   **效率优势**：相比于需要对所有候选三元组进行逐一评分的判别式模型，DiffTSP 的“一次通过（One-pass）”生成模式在大规模推理时效率更高。

### 7. 优点
*   **范式创新**：将 TSP 任务从“独立评分”转向“生成式建模”，从根本上解决了三元组间依赖性难以捕捉的问题。
*   **结构感知设计**：结合了 GNN 的局部提取能力和 Transformer 的全局建模能力，专门适配了多关系图的异构性。
*   **离散扩散适配**：相比连续扩散模型，离散扩散更符合知识图谱这种本质上是离散符号结构的特性。

### 8. 不足与局限
*   **实体固定假设**：模型假设实体集合是预先给定的，无法直接处理推理阶段出现的新实体（Inductive 场景）。
*   **子图依赖**：为了处理大规模图，模型依赖于子图划分策略，这可能导致跨子图的长程依赖在一定程度上被削弱。
*   **计算开销**：虽然推理速度快，但扩散模型的迭代采样过程（如 20 步去噪）相比于单次前向传播的简单模型仍有额外的计算成本。
